Baixar Aplicativo

Este site é uma Progressive Web App (PWA). Você pode instalar o aplicativo diretamente do navegador.

Se desejar um APK para Android, gere um build usando um contêiner que empacote o app web (ex: PWABuilder, bubblewrap).

Abrir App no Navegador Baixar Manifest

Gerar APK

Para gerar um APK utilizável você pode usar ferramentas como:

  • PWABuilder (https://www.pwabuilder.com)
  • Bubblewrap (https://github.com/GoogleChromeLabs/bubblewrap)
  • Capacitor + Ionic (https://capacitorjs.com)
